# BJ (Baekjoon Solutions)

[![Solved.ac Profile](http://mazassumnida.wtf/api/v2/generate_badge?boj=yusengha)](https://solved.ac/yusengha/)

백준 온라인 저지(Baekjoon Online Judge)에서 해결한 문제들의 **정답 코드 아카이브 레포지토리**입니다.  
문제 풀이 기록을 체계적으로 관리하고, 학습 이력을 남기기 위해 운영하고 있습니다.

---

## 📌 Repository Purpose

- 백준 문제 풀이 코드 자동 관리
- 알고리즘 학습 과정 기록
- 티어별 성장 과정 추적
- 문제 풀이 이력의 장기 보관

> ⚠️ 본 레포지토리는 문제 설명이나 해설을 포함하지 않으며,  
> **정답 코드 관리 및 기록**에 초점을 둡니다.

---

## 🤖 How This Repository Is Managed

이 레포지토리는 **BaekjoonHub 확장 프로그램**을 통해 관리됩니다.

- 백준에서 문제 제출 → **정답(AC)** 판정 시
- 문제 번호와 이름으로 폴더 자동 생성
- 해당 폴더 내부에 정답 코드 자동 업로드
- 수동 커밋 없이 GitHub에 반영

👉 모든 코드는 **자동화된 방식**으로 추가됩니다.

---

## 🗂 Directory Rule (Structure Policy)

전체 구조는 문제 수가 많고 지속적으로 증가하기 때문에  
README에 고정된 트리 형태로 나열하지 않습니다.

대신, 다음과 같은 **규칙**을 따릅니다:

- 최상위 폴더: `백준`
- 난이도(Tier) 기준으로 분류
  - Bronze
  - Silver
  - Gold
  - Platinum(아직 플레 레벨의 문제를 풀지 못해 지금은 경로가 없습니다.)
- 각 문제는 다음 형식의 폴더로 관리됩니다:

문제번호_문제이름/


이 규칙은 **BaekjoonHub의 자동 생성 규칙**을 따릅니다.
